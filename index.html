Type Hinting and Return Types

Good Practice 
public function getUserData(int $userId): ?User
{
    return $this->userRepository->findById($userId);
}

Always use type hints and return types for better code clarity and maintainability.

Avoid 
function get_user_data($id) {
    return $this->user_repository->find_by_id($id);
}

Avoid omitting type hints and return types as it makes code harder to understand and maintain.

Important Note

Type hinting helps catch type-related errors early and improves IDE support for better development experience.
